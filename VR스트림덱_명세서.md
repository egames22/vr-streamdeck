## 프로젝트 개요

- **프로그램명:** VR스트림덱.exe

## Dock (메인 창) 명세

| 항목 | 내용 |
|------|------|
| 위치 | 자유롭게 드래그 이동 가능 |
| 가장자리 흡착 | 화면 하단/좌측/우측에 자석처럼 붙음 |
| 자동 숨김 | 가장자리에 붙었을 때 마우스 가져가면 나타남, 멀어지면 숨김 |
| 최상단 고정 | 핀 버튼으로 ON/OFF (어떤 화면에서도 최상단 유지) |
| 크기 조절 | 사용자가 직접 조절 가능 |
| 배열 방식 | 설정에서 선택 (2×3, 3×4 등 예시 표시하여 선택) |
| 투명도 | 사용자 조절 가능 (슬라이더) |
| 테마 | 사용자가 직접 색상 선택 |
| 포커스 | 아이콘 클릭해도 기존 작업창 포커스 유지 (Dock이 포커스 가져가지 않음) |

## 버튼 명세

### 아이콘 표시
| 항목 | 내용 |
|------|------|
| 기본 | 이모지 또는 텍스트 |
| 이미지 | 사용자가 이미지 파일 삽입 시 해당 이미지 표시 |
| 라벨(버튼 이름) | 사용자 선택 (항상 표시 / 호버 시 툴팁 / 숨김) |

### 버튼 기능 목록 (7가지)

| 기능 | 설명 |
|------|------|
| 텍스트 붙여넣기 | 상용구 저장 후 마우스 위치 작업창에 자동 입력 |
| 폴더 열기 | 지정 폴더를 탐색기로 열기 |
| 파일 열기 | 지정 파일 열기 |
| URL 열기 | 지정 웹주소를 기본 브라우저로 열기 |
| 프로그램 실행 | .exe 경로 또는 바로가기 파일로 프로그램 실행 |
| 폴더형 버튼 | 클릭 시 하위 버튼 펼쳐지는 그룹 버튼 |
| 키보드 단축키 | 지정 단축키 실행 (예: Ctrl+Z, Win+D 등) |

### 텍스트 붙여넣기 방식
- 버튼별로 설정 시 선택 가능
- **A안:** 직접 타이핑 (한 글자씩 입력 시뮬레이션)
- **B안:** 클립보드 붙여넣기 (Ctrl+V 시뮬레이션)

### 폴더형 버튼 동작
| 항목 | 내용 |
|------|------|
| 펼침 방향 | Dock 위치 기준 자동 계산 (하단→위, 좌측→오른쪽 등) |
| 닫기 | 다른 곳 클릭 시 닫힘 + 폴더 버튼 재클릭 시 닫힘 |

### 키보드 단축키 입력 방식
- 설정 시 버튼 클릭 후 키보드로 직접 눌러서 녹화 (예: Ctrl+Z 누르면 자동 저장)

## 설정 화면 명세

### 진입 방법
- Dock 우측 톱니바퀴 아이콘 클릭
- Dock 빈 공간 우클릭 메뉴

### 버튼 편집
- 아이콘 우클릭 → 빠른 편집 메뉴
- 설정 화면에서 상세 편집

### 설정 항목
| 항목 | 내용 |
|------|------|
| 테마 색상 | 사용자 직접 선택 |
| 투명도 | 슬라이더 |
| 배열 방식 | 그리드 선택 (2×3, 3×4 등 미리보기 표시) |
| 라벨 표시 | 항상 / 호버 / 없음 선택 |
| Windows 시작 시 자동 실행 | 체크박스 |
| 설정 내보내기 | 버튼 클릭 시 파일 저장 |
| 설정 불러오기 | 버튼 클릭 시 파일 열기 |
